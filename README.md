## About Laravel

Laravel is a web application framework with expressive, elegant syntax. We believe development must be an enjoyable and creative experience to be truly fulfilling. Laravel takes the pain out of development by easing common tasks used in many web projects, such as:

- [Simple, fast routing engine](https://laravel.com/docs/routing).
- [Powerful dependency injection container](https://laravel.com/docs/container).
- Multiple back-ends for [session](https://laravel.com/docs/session) and [cache](https://laravel.com/docs/cache) storage.
- Expressive, intuitive [database ORM](https://laravel.com/docs/eloquent).
- Database agnostic [schema migrations](https://laravel.com/docs/migrations).
- [Robust background job processing](https://laravel.com/docs/queues).
- [Real-time event broadcasting](https://laravel.com/docs/broadcasting).

Laravel is accessible, powerful, and provides tools required for large, robust applications.

# Smart Canteen Pre-Order System

## 📖 Project Overview
The Smart Canteen Pre-Order System is a web-based application that allows students to browse the canteen menu, place food orders online, and reduce waiting time during busy hours. Canteen staff can manage food items and process incoming orders efficiently.

## 🎯 Objectives
- Reduce queues in the university canteen.
- Allow students to pre-order meals.
- Simplify food and order management for canteen staff.

## ✨ Features

### Student
- Register and Login
- Browse Food Menu
- Search Food Items
- Add Food to Cart
- Place Orders
- View Order Status

### Staff/Admin
- Login
- Add/Edit/Delete Food Items
- Manage Customer Orders
- Update Order Status

## 🛠️ Technology Stack
- Frontend: HTML, CSS, Bootstrap
- Backend: PHP
- Database: MySQL
- Version Control: Git & GitHub
- Project Management: Jira

## 🚀 Installation

1. Clone the repository.
2. Import the SQL database into MySQL.
3. Configure the database connection.
4. Place the project inside the XAMPP `htdocs` folder.
5. Start Apache and MySQL.
6. Open the project in your browser.
